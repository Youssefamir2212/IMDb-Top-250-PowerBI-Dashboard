# IMDb Top 250 Movies Dashboard

## Overview
This Power BI dashboard analyzes the IMDb Top 250 Movies dataset to uncover trends in ratings, popularity, duration, release decades, and movie classifications.

## Features

- Average IMDb Rating
- Average Movie Duration
- Average Views
- Rating & Views by Decade
- Movie Classification Analysis
- Rating vs Views Scatter Plot
- Interactive Filters

## Tools

- Power BI
- Power Query
- DAX
- Data Modeling
